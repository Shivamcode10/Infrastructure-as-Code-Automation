## 📌 Overview

A production-ready DevOps pipeline automating infrastructure provisioning and deployment on AWS EKS using IaC and CI/CD, with scalability, monitoring, logging, alerting, and security.

---

## 🎯 Key Achievements

* 🚀 Automated end-to-end deployment (GitHub Actions)
* ☁️ Scalable deployment on AWS EKS
* 📈 Auto-scaling (HPA)
* 📊 Prometheus + Grafana monitoring
* 📜 Centralized logging (Loki)
* 🔔 Slack alert integration
* 🔐 IAM + GitHub Secrets security
* 📦 Helm-based deployments

---

## 🛠️ Tech Stack

**Cloud:** AWS (EKS, EC2, IAM, VPC)
**DevOps:** GitHub Actions, YAML
**IaC:** Terraform
**Deployment:** Kubernetes, Helm
**Monitoring:** Prometheus, Grafana
**Logging & Alerts:** Loki, Slack

---

## ⚙️ Architecture

GitHub → GitHub Actions → AWS IAM → EKS → Kubernetes → Monitoring → Slack Alerts

---

## ✨ Features

* CI/CD automation (push → deploy)
* Secure AWS + EKS integration
* Kubernetes deployment (Deployments, Services, Ingress)
* Helm reusable configs
* HPA auto scaling
* Ingress + HTTPS routing
* Monitoring dashboards (Grafana)
* Centralized logging (Loki)
* Slack alert notifications
* Secure secrets & IAM

---

## 🚀 Workflow

1. Push code
2. GitHub Actions triggers
3. AWS auth configured
4. EKS kubeconfig updated
5. Kubernetes + Helm deployed
6. Monitoring + logging active
7. Slack alerts triggered
